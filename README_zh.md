## 仓库结构

- **void/** - 原始 [Void Editor](https://github.com/voideditor/void) 仓库的Git子模块
- **learning_notes/** - 我个人关于Void的AI代理系统的笔记和学习材料

本仓库作为我学习Void编辑器AI代理实现的个人学习日志。这些笔记记录了我对以下方面的理解：

- 工具系统实现
